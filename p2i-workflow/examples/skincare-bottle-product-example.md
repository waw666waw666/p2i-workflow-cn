# 示例：护肤品瓶身电商主图

## 粗略想法

```txt
做一张高级感护肤精华瓶产品图，干净背景，玻璃瓶身，柔和高光，像电商主图和品牌广告图。
```

## 提示词2示例

【图片类型判断】  
这属于产品图 / 电商主图 / 商业广告视觉方向。核心目标是突出护肤精华瓶的材质、光影和高级但克制的品牌感。

【参考方向】  
参考商业护肤品产品图、电商主图、品牌棚拍广告类提示词结构，重点突出单一主体、玻璃材质、留白背景、可控高光和干净构图。

【结构提炼】  
主体 -> 单支护肤精华瓶  
材质 -> 玻璃瓶身、半透明液体、金属或磨砂瓶盖  
场景 -> 极简棚拍背景 / 干净台面  
构图 -> 居中主构图，大面积留白  
镜头 -> 近景产品摄影  
光影 -> 柔和棚拍光、边缘高光、轻反射  
色彩 -> 低饱和中性色 / 高级护肤品配色  
风格 -> 商业广告、电商主图、极简高级感  
质量 -> 高清细节，瓶身边缘干净锐利  
用途 -> 电商主图、品牌 KV、详情页首屏

【最终可复制提示词】
```txt
中文最终提示词：
创作一张高级克制风格的护肤精华瓶产品图，主体是一支单独摆放的护肤精华瓶，瓶身为透明或半透明玻璃材质，内部可见轻微通透感的精华液体，瓶盖可以是磨砂金属或简洁高级塑料材质，整体造型现代、精致、比例协调。瓶身边缘有干净利落的反光线条，玻璃表面有柔和高光和细腻反射，标签区域简洁，不出现杂乱文字和多余图形。

场景设置为极简商业棚拍环境，背景干净统一，可以是浅米白、奶油白、浅灰或淡暖色调，搭配平整纯净台面或轻微倒影面，不加入复杂道具，最多保留少量克制的水波纹光影、柔和投影或半透明亚克力结构，用来增强高级护肤广告氛围。构图采用居中主构图或轻微偏中构图，主体占比清晰，保留大面积干净留白，适合品牌广告和电商主图使用。

光影使用柔和商业棚拍光，正面柔光保证瓶身轮廓清楚，侧边加入细窄边缘高光突出玻璃质感，局部保留自然反射和柔和阴影，整体明亮、干净、克制，不要过度戏剧化打光。整体风格为高端护肤品商业广告、电商主图、极简产品摄影，强调玻璃质感、纯净背景、精致细节和可信的产品呈现，高清细节，4:5 画幅。

Negative Prompt：
low quality, blurry, noisy, cluttered background, messy props, distorted bottle, warped glass, label error, garbled text, watermark, logo duplication, dirty surface, scratched bottle, overexposed highlights, harsh shadow, flat lighting, bad reflections, cheap packaging look, crowded composition
```

【可替换变量】  
- 主体：精华瓶 / 面霜罐 / 喷雾瓶 / 安瓶 / 化妆水瓶  
- 风格：极简高级 / 电商主图 / 品牌广告 / 医美感 / 清透自然  
- 颜色：米白 / 浅灰 / 奶油白 / 淡粉 / 淡金  
- 场景：纯净棚拍背景 / 亚克力台面 / 轻倒影台面 / 浅色渐变背景 / 柔光背景墙  
- 镜头：正面近景 / 微俯视 / 居中特写 / 侧前方产品角度 / 细节局部特写  
- 光影：柔和棚拍光 / 侧边缘高光 / 自然窗感柔光 / 水波纹反射 / 干净商业打光  
- 材质：透明玻璃 / 磨砂玻璃 / 金属瓶盖 / 亚克力包装 / 半透明液体  
- 构图：居中构图 / 偏中构图 / 大留白广告图 / 对称构图 / 详情页首图构图  
- 画幅：4:5 电商图 / 1:1 方图 / 9:16 竖版 / 16:9 横幅  
- 用途：电商主图 / 品牌 KV / 详情页首屏 / 社媒广告图 / 海报视觉

【可选版本方向】  
1. 更电商主图版：减少氛围元素，强化清晰度和商品识别。  
2. 更品牌广告版：增加克制的光影设计和留白感。  
3. 更医美科技版：适合功能型护肤品，强化理性、洁净、专业感。  
4. 更自然清透版：适合补水类产品，增强柔和透亮氛围。  
5. 更高奢版：适合高端线产品，强化金属细节和精致反射。

【质量检查】  
- 主体是否明确：明确，主体是一支单独护肤精华瓶。  
- 风格是否统一：统一为极简高级产品广告风格。  
- 是否有互斥描述：没有，未混入复杂场景或多主体。  
- 是否适合 B窗口直接复制：适合。  
- 是否建议进一步指定变量：建议指定瓶身颜色和瓶盖材质。  
- 最该补充的 1 个信息：品牌想要偏医美感还是偏生活方式高级感。
