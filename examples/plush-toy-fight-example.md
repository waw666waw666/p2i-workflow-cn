# 示例：可爱毛绒玩偶打架场景

```txt
两个毛绒玩偶在迷你桌面擂台上打架，可爱幽默，不血腥，像玩具广告图。
```

## 提示词2示例

【图片类型判断】  
这属于玩具摄影 / 趣味广告视觉 / 3D 卡通质感方向。核心目标是把“打架”表达成可爱幽默、无血腥、适合传播的玩具广告场景。

【参考方向】  
参考商业玩具摄影、3D 卡通产品图、趣味社交媒体封面类提示词结构，重点突出主体动作、毛绒材质、迷你场景、柔和光影和轻喜剧气质。

【结构提炼】  
主体 -> 两个毛绒玩偶  
材质 -> 柔软毛绒、布料褶皱、塑料眼睛  
场景 -> 迷你桌面擂台 / 玩具小舞台  
构图 -> 低角度近景，左右对峙  
镜头 -> 近景产品摄影  
光影 -> 柔和棚拍光，暖色边缘光  
色彩 -> 明亮、温暖、可爱
